# Ecosia Bot

Hi, I saw a tiktok of someone using Ecosia so I made a bot for it

# Requirements

**PYTHON**

# Installation

Clone the repo
```
git clone https://github.com/delltaxa/ecosia-bot.git
```

Go in to the directory
```
cd ecosia-bot/
```

Add perms to run farm.sh
```
chmod +x ./farm.sh
```
